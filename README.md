# 🌴 Island Survival Game

A simple text-based survival game built using Python, where you try to survive on a deserted island.

## 🎮 Features
- Resource management (health, etc.)
- Decision-based gameplay
- Survival mechanics
- Event-driven system

## 🛠 Tech Used
- Python

## 🚀 How to Run
1. Download the project files  
2. Run `main.py`  
3. Follow the prompts to survive  

## 📌 Notes
This is a beginner project made to practice Python and game logic.

Future improvements may include more events, better mechanics, and expanded gameplay.
